## 🛠️ 스터디 룰

| 항목 | 내용 |
| :--- | :--- |
| **문제 선정** | [기본 문제집](https://github.com/encrypted-def/basic-algo-lecture/blob/master/workbook.md) 기준 매주 **기본 1문제 + 응용 2문제** |

<br>

## 🚀 제출 방법

1. 초기 설정 및 업데이트
* **최초 1회:** 본 저장소를 `fork` 한 후, 내 컴퓨터로 `clone` 합니다.
* **매주 업데이트:** 새로운 주차 폴더가 생기면 `pull` 또는 `fetch`를 통해 최신 상태를 유지합니다.

2. 코드 제출 (`commit` & `push`)
* 각 주차 폴더에 소스코드를 저장합니다.
* **파일명 규칙:** `문제번호_이름.java` (ex. `12345_홍길동.java`)

3. PR (Pull Request) 보내기
* PR 제목은 아래 규칙을 엄수합니다.
* **제목 규칙:** `[이름] O주차 (문제번호) 과제 제출`
* **예시:** `[홍길동] 1주차 12345번 과제 제출` (일부만 풀었을 경우 번호 명시)

4. 피드백 및 머지
* `changes requested` 상태라면 리뷰어의 피드백을 반영하여 코드를 수정 후 다시 `push` 합니다.
* 모든 피드백이 반영되어 승인되었다면 `merged` 후 해당 주차를 마무리합니다.
